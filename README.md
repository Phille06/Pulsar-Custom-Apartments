# Pulsar-Custom-Apartments
This is a completely rewritten Apartment Resource for Pulsar Framework. It offers the following;

# Features
- Walk-in Apartments with integrated Police Raiding.
- Each Apartment Floor is Seperated Buckets, you can see everyone on your floor.
- It comes Set up with Kiiya's La Puerta Apartments as well as the free Wiwang Building.
- Players can choose where they want to live upon first spawn.
- UI pulls from unassigned apartments to give a live count on how many rooms are available in each apartment building.
- Each week players get charged $2,000 for rent via Pulsar-Finance (directly one week after assignment).
